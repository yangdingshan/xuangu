# A股选股器 — 设计文档

**日期**: 2026-05-20
**状态**: 待实施

## 概述

一个基于 Web 的 A 股实时选股筛选器。用户打开页面，点击筛选，系统通过 AKShare 获取全市场实时数据，按预设条件过滤，展示符合条件的股票列表。

## 筛选条件（可配置，每个条件可独立开关）

每个筛选条件由三部分组成：**开关**（启用/禁用）、**参数值**（可调整）、**标签**（展示当前设置）。

| 条件 | 默认值 | 参数 | 说明 |
|------|--------|------|------|
| 涨幅 | 启用 | 3% ~ 5% | 当日涨幅区间，可分别调整下限和上限 |
| 涨停历史 | 启用 | 近20日，至少1次 | 回溯天数和涨停次数均可调整 |
| 量比 | 启用 | >= 1 | 最小量比阈值，可调整 |
| 换手率 | 启用 | 5% ~ 10% | 换手率区间，可分别调整下限和上限 |
| 市值 | 启用 | 50亿 ~ 200亿 | 市值区间，可分别调整下限和上限 |

**交互规则**：
- 每个条件左侧有开关，关闭后该条件不参与筛选
- 条件关闭后参数输入框置灰不可编辑
- 默认全部启用，参数使用预设默认值
- 用户修改的参数在页面刷新后保留（localStorage）

## 技术方案

- **后端**: Python FastAPI（异步）
- **前端**: 原生 HTML + CSS + JavaScript（无框架）
- **数据源**: AKShare（免费开源 Python 库）
- **运行方式**: 手动触发表单次筛选

## 页面结构

### 筛选条件栏
- 横向排列五组筛选条件，每组包含：
  - 开关 toggle（左侧，控制该条件是否启用）
  - 条件名称标签
  - 参数输入框（数值范围或单值，条件关闭时置灰）
- 右侧「开始筛选」按钮（teal 色主按钮）
- 用户修改的参数自动保存到 localStorage，刷新后恢复

### 统计摘要行
- 四列指标卡片：扫描总数、符合条件数、更新时间（含实时指示器）、选中率

### 结果表格
- 经典紧凑表格，浅色主题
- 列：代码、名称、现价、涨幅、量比、换手率、市值、近20日涨停次数、行业
- 功能色标识：涨幅红色、涨停橙色、代码蓝色可点击
- 深色表头、行间交替色、hover 高亮
- 右上角「导出」「自选」操作按钮

### 状态覆盖
- **加载态**: 骨架屏（shimmer 动画），不是转圈
- **空结果态**: 图标 + 说明文字 + 建议调整条件
- **异常态**: 红色左边框错误卡片 + 错误描述 + 重试按钮

## 数据流

```
用户点击筛选
  → 前端收集已启用的筛选条件及参数
  → 前端显示骨架屏
  → POST /api/screen (携带筛选条件 JSON)
    → AKShare 获取全市场实时行情
    → AKShare 获取近20日涨停数据（仅在涨停条件启用时）
    → 按已启用的条件过滤
    → 返回 JSON 结果列表
  → 前端渲染表格
```

## API 设计

### `POST /api/screen`
提交筛选条件，触发一次全市场筛选，返回符合条件的股票列表。

请求体：
```json
{
  "filters": {
    "change_pct": { "enabled": true, "min": 3, "max": 5 },
    "limit_up": { "enabled": true, "days": 20, "min_count": 1 },
    "volume_ratio": { "enabled": true, "min": 1 },
    "turnover_rate": { "enabled": true, "min": 5, "max": 10 },
    "market_cap": { "enabled": true, "min": 50, "max": 200 }
  }
}

响应：
```json
{
  "total_scanned": 5263,
  "matched": 12,
  "updated_at": "14:32:05",
  "stocks": [
    {
      "code": "000625",
      "name": "长安汽车",
      "price": 18.52,
      "change_pct": 4.32,
      "volume_ratio": 1.28,
      "turnover_rate": 7.21,
      "market_cap": 138.5,
      "limit_up_count": 2,
      "sector": "汽车整车"
    }
  ]
}
```

### `GET /`
返回静态页面（index.html）。

## 依赖

- `fastapi` + `uvicorn` — Web 框架
- `akshare` — A 股数据
- `pandas` — 数据处理

## 非目标

- 不做盘中自动刷新（后续迭代）
- 不做用户登录/自选股持久化
- 不做 K 线图/技术分析图表
- 不做移动端适配（桌面优先）
