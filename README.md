# A 股选股器

基于 FastAPI + 原生 HTML/CSS/JS 的 A 股筛选工具，支持 5 个可独立开关的筛选条件。

## 启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
python -m uvicorn main:app --host 0.0.0.0 --port 8000

# 3. 浏览器打开
# http://localhost:8000
```

## 筛选条件

| 条件 | 默认值 | 说明 |
|---|---|---|
| 涨幅 | 3% ~ 5% | 当日涨跌幅 |
| 涨停 | 近 20 日 >= 1 次 | 回溯涨停次数（含 10%/20%/30% 涨跌停板） |
| 量比 | >= 1 | 成交量与近期均量之比 |
| 换手率 | 5% ~ 10% | 当日换手率 |
| 市值 | 50 ~ 200 亿 | 总市值（人民币） |

每个条件可独立开关并调整参数，配置自动保存到 localStorage。
